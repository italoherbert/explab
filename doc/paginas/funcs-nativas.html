Funções globais predefinidas

abs

abs( numerico ) : numerico

A função abs retorna um valor real se receber um valor real como parâmetro e retorna um valor tipo matriz se receber um valor tipo matriz como parâmetro.

Exemplo de uso:


a = abs( -1 );  
b = abs( 1 );	   
						   	

exibaln abs( [ -1, 2, -3 ] );   // exibe: 1 2 3

exibaln abs( [ -1 2 -3] );      // exibe: 1 1  
                                // Isso porque sem as vírgulas o interpretador entende que os dois 
                                // ultimos elementos compõem uma expressão numérica e, então, 
                                // realiza a subtração: 2 -3 que corresponde a -1.
exibaln a;
exibaln b;

No código acima a variável "a" recebe o módulo de -1 que corresponde ao valor 1 e a variável "b" recebe o módulo de 1 que é 1 mesmo. Logo após, é exibido o resultado da função abs aplicada a um valor matriz. Perceba que os elementos estão separados por virgula! Já na chamada ao comando exibaln seguinte, a função abs recebe uma matriz que difere da primeira matriz, apesar de se poder separar os elementos das matrizes, tanto com espaços quanto com vírgulas.

acos

acos( numerico ) : numerico

A função acos retorna o arco cosseno (em radianos) de um valor real, se receber um valor real como parâmetro e retorna um valor tipo matriz, se receber um valor tipo matriz de valores reais em radianos como parâmetro. A função acos é inversa a função cos, isto é, se calcularmos o cos de um valor e este valor for passado para a função acos, o resultado é o mesmo ângulo em radianos passado para a função cos.

Exemplo de uso:


a = acos( cos( pi ) );
exibaln a;   // exibe o valor correspondente a constante pi: 3,14159....

atan2

atan2( numerico, numerico ) : numerico

A função atan2 retorna o arco tangente (em radianos) de dois valores reais: O valor de "y" (cateto oposto) e o valor de "x" (cateto adjacente). Isto é, a função tan retorna y/x e a função atan2 retorna atan( y/x ).

Exemplo de uso:


x1 = 10;
y1 = 10;
x2 = 100;
y2 = 80;
a = atan2( y2-y1, x2-x1 );  // a recebe 0,661. Este valor está em radianos

exibaln a;                  // exibe o valor de a em radianos
exibaln graus( a );         // exibe o valor de a em graus: 37,875
exibaln atan( 40/10 ) == atan2( 40, 10 );   // exibe o valor verdade

plot

plot( PC2D )

A função plot pode ser utilizada em conjunto com as classes PC2D, Dados2D e Func2D, entre outras classes de que estas são compostas. Sua principal funcionalidade é a plotagem de funções e/ou dados em 2D. Recebe como parâmetro um objeto instância da classe PC2D e plota um gráfico num plano cartesiano conforme a configuração definida a partir da classe PC2D e suas classes componentes.

Exemplo de uso:


x=vetesp( -pi, pi, 300 );
y=cos(x);

d = novo Dados2D();
d.xvetor = x;
d.yvetor = y;

pc = novo PC2D();

pc.dados[] = d;

plot( pc );

A função vetesp, no exemplo acima, gera 300 valores no intervalo de -pi a pi e os armazena em x. Em seguida a variável y recebe a matriz unidimensional com os cossenos dos valores de x. Depois, a classe Dados2D é instanciada e armazenada no objeto d. Os atributos do objeto d (xvetor e yvetor) são setados com os valores das variáveis x e y. Então, é instanciada a classe PC2D no objeto pc. O objeto pc tem como atributo um vetor para armazenar objetos da classe Dados2D, chamado de "dados". Então, o objeto "d" é adicionado ao vetor de dados para, logo em seguida, a função plot receber o objeto pc como parâmetro e gerar o gráfico correspondente a configuração do objeto pc e seus componentes.

plot3d

plot3d( PC3D )

A função plot3d pode ser utilizada em conjunto com as classes PC3D, Dados3D e Func3D, Superficie3D, entre outras classes de que estas são compostas. Sua principal funcionalidade é a plotagem de funções e/ou dados em 3D. Recebe como parâmetro um objeto instância da classe PC3D e plota um gráfico num plano cartesiano tridimensional conforme a configuração definida a partir da classe PC3D e suas classes componentes.

Exemplo de uso:


x = vetesp( -pi, pi, 20 );
y = x;
xx = xmalhagrade( x, y );
yy = ymalhagrade( x, y );
z = sen( sqrt( xx^2 + yy^2 ) );

superficie = novo Superficie3D();
superficie.setDados( x, y, z );   // poderia-se utilizar os atributos xvetor, yvetor, zmat

pc = novo PC3D();
pc.dados[] = superficie;

plot3d( pc );

A função vetesp, no exemplo acima, gera 20 valores no intervalo de -pi a pi e os armazena em x. Em seguida a variável y recebe uma cópia da matriz unidimensional x. Depois disto, as classes xmalhagrade e ymalhagrade são utilizadas para gerar matrizes que possam ser manipuladas com as funções numéricas e, assim, gerar a matriz correspondente a necessária ao atributo zmat da classe Superficie3D. Depois, a classe Superficie3D é instanciada e armazenada em "superficie" e o método setDados é chamado para setar os valores x, y e z. Então, o objeto pc recebe a instância da classe PC3D e seu atributo dados, que corresponde a um vetor de elementos do tipo Dados3D ou Superficie3D, é acrescido do objeto "superficie". Em seguida, a função plot recebe o objeto pc como parâmetro e gera o gráfico correspondente a configuração do objeto pc e seus componentes.

ymalhagrade( matriz, matriz ) : matriz

A função ymalhagrade pode ser utilizada em conjunto com a plot3d. Ela recebe como parâmetro dois vetores - matrizes unidimensionais. (se o número de linhas for maior que 1, apenas a primeira linha é considerada). A matriz resultante da função ymalhagrade, tem como número de linhas o número de elementos do segundo vetor (ou número de colunas se for matriz com mais que 1 linha) e como número de colunas da matriz resultante tem como valor o número de elementos (ou colunas) do primeiro vetor. Os elementos do segundo vetor são repetidos em todas as colunas da matriz resultante.

Exemplo de uso:


x = vetesp( 0, 4, 5 );    // gera uma matriz unidimensional com os valores: 0,1,2,3 4
y = vetesp( 0, 6, 7 );    // gera uma matriz unidimensional com os valores: 0,1,2,3,4,5,6
yy = ymalhagrade( x, y )  // gera uma matriz com o vetor "y" repetido 5 vezes nas colunas da matriz "yy".
                          // Isso porque o número de elementos do vetor x corresponde a 5

zeros( real, real ) : matriz

A função zero gera uma matriz de zeros de dimensão MxN, isto é, o primeiro parâmetro representa o "M", e o segundo, o "N".

Exemplo de uso:


a = zeros( 2, 3 );           // gera a matriz [ 0 0 0; 0 0 0 ]
b = preenche( 2, 3, 1 );     // nesse caso faz o mesmo que a função uns

exibaln a == b;    // exibe verdade

Classe PoligonoConfig

Esta classe pode ser utilizada em conjunto com a classe PC3D e Superficie3D para configurar os polígonos no plano cartesiano 3D que compõem uma superfície.

Atributos

Nome Tipo Valor padrão Descrição
visivel boolean verdade Variável que armazena um valor booleano que, se verdade, configura os polígonos para visível, se falso, configura os polígonos para não visível
cor RGB preto Objeto da classe RGB que permite a alteração e configuração da cor dos polígonos
gradiente boolean verdade Variável que armazena um valor booleano que, se verdade, configura para que os polígonos sejam pintados não com a cor especificada, mas com um gradiente que varia da cor azul a vermelho. Quanto mais alto o polígono em relação ao eixo Z, mais próximo do vermelho, quanto mais baixo o polígono em relação ao eixo Z, mais próximo da cor azul.
iluminar boolean verdade Variável que armazena um valor booleano que, se verdade, configura as linhas terem iluminação aplicada a cor delas. Se falso, configura as linhas para serem pintadas com a cor do atributo cor (sem iluminação)

Construtores

Esta classe não possue construtores definidos

Funções

Esta classe não tem funções definidas

Exemplo de uso


poligono = novo PontoConfig;
poligono.visivel = verdade;   // polígonos visíveis
poligono.cor.r = 255;         // quantidade de cor vermelha alterada
poligono.cor.g = 100;         // quantidade de cor verde alterada
poligono.cor.b = 0;           // sem quantidade de cor azul
poligono.gradiente = falso;   // sem efeito gradiente
poligono.iluminar = falso;    // sem efeito iluminação

poligono2 = novo PontoConfig;
poligono2.cor = novo RGB( 0, 255, 0 );  // polígonos configurados com a cor verde

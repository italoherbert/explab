Classe GradeConfig

Esta classe pode ser utilizada em conjunto com as classes PC2D e PC3D para configurar a grade do plano cartesiano que será mostrado, após a chamada da função plot ou plot3d.

Atributos

Nome Tipo Valor padrão Descrição
visivel boolean verdade Variável que armazena um valor booleano que, se verdade, configura a grade para visível, se falso, configura a grade para não visível
cor RGB preto Objeto da classe RGB que permite a alteração e configuração da cor da grade

Construtores

Esta classe não possue construtores definidos

Funções

Esta classe não tem funções definidas

Exemplo de uso


grade = novo GradeConfig;
grade.visivel = verdade;   // grade visível
grade.cor.r = 255;         // quantidade de cor vermelha alterada
grade.cor.g = 100;         // quantidade de cor verde alterada
grade.cor.b = 0;           // sem quantidade de cor azul

grade2 = novo GradeConfig;
grade2.cor = novo RGB( 0, 255, 0 );  // grade configurada com a cor verde

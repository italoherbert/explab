Classe Func2D

Esta classe pode ser utilizada em conjunto com a classe PC2D e o comando plot para plotagem de um gráfico com base em intervalos de x e/ou de y e a definição de uma função matemática com expressão numérica de retorno para o calculo dos valores de y.

Atributos

Nome Tipo Valor padrão Descrição
x1 real -1 Extremidade inferior do intervalo de x
x2 real 1 Extremidade posterior do intervalo de x
y1 real -1 Extremidade inferior do intervalo de y
y2 real 1 Extremidade posterior do intervalo de y
limitarY boolean falso ativa ou desativa o intervalo de y. Se desativado, o intervalo de y é calculado automaticamente, sem limita-lo (ou expandi-lo) aos valores y1 e y2
xcompleto boolean falso ativa ou desativa que o intervalo de x da função herde do intervalo de x do plano cartesiano, mesmo que este seja maior que o intervalo de x configurado para a função
fun func func( x ) { retorne x } É neste atributo que deve ser setado a variável função (A função pode ser definida diretamente).
legenda string null Texto com a legenda associada a função a ser plotada. Se valer null, a legenda é omitida no gráfico
ponto PontoConfig um objeto Permite a configuração dos pontos de cada dado. É possível configurar a visibilidade, cor e ráio da forma que é pintada se a visibilidade estiver ativada
linha LinhaConfig um objeto Permite a configuração das linhas que ligam cada dado. É possível configurar a visibilidade e a cor

Construtores

construtor()

Se a classe for instanciada com o construtor vasio, os valores padrão são assumidos.

construtor( real, real )

Esse construtor recebe como parâmetro os valores do intervalo de x (x1 e x2) e os seta nas devidas variáveis.

construtor( real, real, func )

Este construtor recebe como parâmetro os valores do intervalo de x (x1 e x2) e uma variável tipo função. Os valores são setados nas devidas variáveis

Funções

Esta classe não tem funções definidas

Exemplo de uso

Esse exemplo plota o gráfico da função cosseno com x variando no intervalo de -pi a pi. No entanto, foi utilizada a classe Func2D, ao invés de se optar por gerar os dados correspondentes aos pontos que compõem a função e utilizar a classe Dados2D. O objeto Func2D foi configurado com mais flexibilidade do que com a utilização da classe Dados2D.


f = novo Func2D( -pi, pi );             // instancia a classe Func2D com os intervalos de X  
f.y1 = -2.0;                            // seta o primeiro intervalo de y
f.y2 = 2.0;                             // seta o segundo intervalo de y
f.limitarY = verdade;                   // Com esse atributo setado para verdade, o intervalo 
                                        // de y será considerado na construção do gráfico

f.xcompleto = verdade;                  // Com esse atributo setado para verdade, a função 
                                        // é desenhada de acordo com o intervalo de x do 
                                        // plano cartesiano, mesmo que este seja maior que os 
                                        // intervalos de x da função

f.legenda = "cosseno";                  // adiciona uma legenda para a função
f.linha.cor = novo RGB( 0, 0, 255 );    // seta a cor da função no gráfico

f.fun = func( x ) {                     // define a função que será utilizada para o cálculo 
    retorne cos( x );                   // dos valores de y no plano cartesiano
};

pc = novo PC2D();      
pc.graficos[] = f;                      // adiciona o objeto f ao vetor de graficos

plot( pc );                             // plota o gráfico

Outras configurações


f = novo Func2D();            // instancia a classe Func2D
f.ponto.visivel = verdade;    // configura os pontos de dados para visíveis
f.ponto.cor.r = 255;          // altera a porção de cor vermelha do ponto para 255
f.ponto.cor.g = 0;            // altera a porção de cor verde do ponto para 0
f.ponto.cor.b = 0;            // altera a porção de cor azul do ponto para 0 
f.ponto.raio = 3;

f.linha.visivel = verdade;    // configura as linhas que ligam os dados para visíveis
f.linha.cor.r = 0;            // altera a porção de cor vermelha das linhas para 0
f.linha.cor.g = 255;          // altera a porção de cor verde das linhas para 255
f.linha.cor.b = 0;            // altera a porção de cor azul das linhas para 0

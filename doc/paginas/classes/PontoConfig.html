Classe PontoConfig

Esta classe pode ser utilizada em conjunto com as classes PC2D e PC3D para configurar os pontos no plano cartesiano que compõem uma função ou curva representada pelos dados que serão mostrados, após a chamada da função plot ou plot3d.

Atributos

Nome Tipo Valor padrão Descrição
visivel boolean verdade Variável que armazena um valor booleano que, se verdade, configura os pontos para visível, se falso, configura os pontos para não visível
cor RGB preto Objeto da classe RGB que permite a alteração e configuração da cor dos pontos
raio real 5 Variável que permite a configuração do ráio dos círculos ou esferas a serem pintadas como pontos no plano cartesiano

Construtores

Esta classe não possue construtores definidos

Funções

Esta classe não tem funções definidas

Exemplo de uso


ponto = novo PontoConfig;
ponto.visivel = verdade;   // pontos visíveis
ponto.cor.r = 255;         // quantidade de cor vermelha alterada
ponto.cor.g = 100;         // quantidade de cor verde alterada
ponto.cor.b = 0;           // sem quantidade de cor azul
ponto.raio = 3;

ponto2 = novo PontoConfig;
ponto2.cor = novo RGB( 0, 255, 0 );  // pontos configurados com a cor verde

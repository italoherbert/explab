Classe Dados2D

Esta classe pode ser utilizada em conjunto com a classe PC2D e o comando plot para plotagem de um gráfico com base em dados gerados que possam formar uma curva, gráfico de função matemática ou sequência de linhas.

Atributos

Nome Tipo Valor padrão Descrição
xvetor matriz vetor Vetor de valores de x para os dados a serem plotados
yvetor matriz vetor Vetor de valores de y para os dados a serem plotados
legenda string null Texto com a legenda associada aos dados a serem plotados. Se valer null, a legenda é omitida no gráfico
ponto PontoConfig um objeto Permite a configuração dos pontos de cada dado. É possível configurar a visibilidade, cor e ráio da forma que é pintada se a visibilidade estiver ativada
linha LinhaConfig um objeto Permite a configuração das linhas que ligam cada dado. É possível configurar a visibilidade e a cor

Construtores

Esta classe não tem construtores definidos

Funções

setDados( matriz, matriz )

Esta função não tem valor de retorno e recebe como parâmetro os vetores com os valores de x e de y

Exemplo de uso

Esse exemplo plota o gráfico de dados gerados para um vetor x no plano cartesiano e, a esse vetor x, é aplicada a função cosseno para gerar o vetor y. O resultado é uma ilustração da função cosseno com x variando no intervalo de -pi a pi.


x=vetesp( -pi, pi, 300 );    // gera 300 valores de x no intervalo de -pi a pi
y=cos(x);                    // gera os valores de y com aplicação da função 
                             // cosseno sobre os valores de x

d = novo Dados2D();          // instancia a classe Dados2D
d.xvetor = x;                // seta o vetor de x
d.yvetor = y;                // seta o vetor de y
//d.setDados( x, y );        // modo alternativo de setar os vetores x e y

pc = novo PC2D();              // instancia a classe PC2D
pc.graficos[] = d;           // adiciona o objeto d ao vetor de graficos

plot( pc );                  // plota o gráfico

No exemplo acima, a função vetesp é utilizada para gerar 300 valores no intervalo de -pi a pi e os armazenar em x. Em seguida a variável y recebe a matriz unidimensional com os cossenos dos valores de x. Depois, a classe Dados2D é instanciada e armazenada no objeto d. Os atributos do objeto d (xvetor e yvetor) são setados com os valores das variáveis x e y. Então, é instanciada a classe PC2D no objeto pc. O objeto pc tem como atributo um vetor para armazenar objetos da classe Dados2D, chamado de "dados". Então, o objeto "d" é adicionado ao vetor de dados para, logo em seguida, a função plot receber o objeto pc como parâmetro e gerar o gráfico correspondente a configuração do objeto pc e seus componentes. Outras configurações


d = novo Dados2D();           // instancia a classe Dados2D
d.ponto.visivel = verdade;    // configura os pontos de dados para visíveis
d.ponto.cor.r = 255;          // altera a porção de cor vermelha do ponto para 255
d.ponto.cor.g = 0;            // altera a porção de cor verde do ponto para 0
d.ponto.cor.b = 0;            // altera a porção de cor azul do ponto para 0 
d.ponto.raio = 3;

d.linha.visivel = verdade;    // configura as linhas que ligam os dados para visíveis
d.linha.cor.r = 0;            // altera a porção de cor vermelha das linhas para 0
d.linha.cor.g = 255;          // altera a porção de cor verde das linhas para 255
d.linha.cor.b = 0;            // altera a porção de cor azul das linhas para 0

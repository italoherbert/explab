Classe JanelaConfig

Esta classe pode ser utilizada em conjunto com as classes PC2D e PC3D para configurar as dimensões e título da janela onde o gráfico será mostrado, após a chamada da função plot ou plot3d.

Atributos

Nome Tipo Valor padrão Descrição
titulo string ExpLab - Aplicação Variável que armazena o título da janela (tipo string)
largura real 400 Variável que armazena a largura da janela (tipo real - quando setado na configuração da janela, o valor é truncado e convertido para inteiro)
altura real 400 Variável que armazena a altura da janela (tipo real - quando setado na configuração da janela, o valor é truncado e convertido para inteiro)

Construtores

construtor()

Se a classe JanelaConfig for instanciada com o construtor vazio, os valores definidos como padrão são assumidos.

construtor( string, real, real )

Se passados como parâmetro, os atributos da classe são setados com eles. O construtor atribui os valores dos parâmetros aos atributos correspondentes.

Funções

Esta classe não tem funções definidas

Exemplo de uso


janela = novo JanelaConfig();
janela2 = novo JanelaConfig( "Meu título", 640, 480 );  // configura um título personalizado e as 
                                                        // dimensões da janela são setadas para 640x480
														
janela3 = novo JanelaConfig;
janela3.titulo = "Meu titulo";
janela3.largura = 200;
janela3.altura = 300;                                   // janela configurada sem uso do construtor

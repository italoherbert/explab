Classe Func3D

Esta classe pode ser utilizada em conjunto com a classe PC3D e o comando plot3d para plotagem de um gráfico com base em intervalos de x, de y e/ou de z e a definição de uma função matemática com expressão numérica de retorno para o calculo dos valores de z em função dos valores de x e de y.

Atributos

Nome Tipo Valor padrão Descrição
x1 real -1 Extremidade inferior do intervalo de x
x2 real 1 Extremidade posterior do intervalo de x
y1 real -1 Extremidade inferior do intervalo de y
y2 real 1 Extremidade posterior do intervalo de y
z1 real -1 Extremidade inferior do intervalo de z
z2 real 1 Extremidade posterior do intervalo de z
xndivs real 20 Numero de polígonos da malha na direção do eixo x
yndivs real 20 Numero de polígonos da malha na direção do eixo y
limitarZ boolean falso ativa ou desativa o intervalo de z. Se desativado, o intervalo de z é calculado automaticamente, sem limita-lo (ou expandi-lo) aos valores z1 e z2
somenteMalha boolean falso ativa ou desativa o desenho de, apenas as arestas da malha em caso de valer verdade e, em caso de valer falso, o desenho dos polígonos pintados conforme a configuração do objeto "poligono" é feito, formando o gráfico de uma superfície
fun func func( x, y ) { retorne x } É neste atributo que deve ser setado a variável função (A função pode ser definida diretamente).
legenda string null Texto com a legenda associada a função a ser plotada. Se valer null, a legenda é omitida no gráfico
ponto PontoConfig um objeto Permite a configuração dos pontos de cada dado. É possível configurar a visibilidade, cor e ráio da forma que é pintada se a visibilidade estiver ativada
linha LinhaConfig um objeto Permite a configuração das linhas que ligam cada dado. É possível configurar a visibilidade e a cor, os efeitos: gradiente e iluminação para cores.
poligono PoligonoConfig um objeto Permite a configuração dos polígonos da malha gerada para plotagem do gráfico. É possível configurar a visibilidade, a cor e os efeitos: gradiente e iluminação para cores.

Construtores

construtor()

Se a classe for instanciada com o construtor vasio, os valores padrão são assumidos.

construtor( real, real, real, real )

Esse construtor recebe como parâmetro os valores do intervalo de x (x1 e x2) e do intervalo de y (y1 e y2) e os seta nas devidas variáveis.

construtor( real, real, real, real func )

Este construtor recebe como parâmetro os valores do intervalo de x (x1 e x2) e do intervalo de y (y1 e y2) e uma variável tipo função. Os valores são setados nas devidas variáveis.

Funções

Esta classe não tem funções definidas

Exemplo de uso

Exemplo 3: Este exemplo desenha a superfície de uma função matemática que envolve as funções seno e cosseno. É utilizada a classe Func3D e configurado o intervalo, ao invés de os dados serem gerados através das funções vetesp, xmalhagrade e ymalhagrade.


f = novo Func3D();                // instância da classe Func3D

f.x1 = -pi;                       // valor inferior do intervalo de x setado para -pi
f.x2 = pi;                        // valor posterior do intervalo de x setado para pi 
f.y1 = -pi;                       // valor inferior do intervalo de y setado para -pi
f.y2 = pi;                        // valor posterior do intervalo de y setado para pi

f.fun = func( x, y ) {            // definição da função 3D envolvendo dois parâmetros: x e y
    retorne 3*sen( x ) + 8*cos( y );
}

pc = novo PC3D();                 // instância da classe PC3D
pc.graficos[] = f;                // adição do objeto ao vetor de graficos

plot3d( pc );                     // plota o gráfico

Outras configurações


f = novo Func3D();            // instancia a classe Func2D
f.ponto.visivel = verdade;    // configura os pontos de dados para visíveis
f.ponto.cor.r = 255;          // altera a porção de cor vermelha do ponto para 255
f.ponto.cor.g = 0;            // altera a porção de cor verde do ponto para 0
f.ponto.cor.b = 0;            // altera a porção de cor azul do ponto para 0 
f.ponto.raio = 3;             // altera o raio do vértice (ponto) para 3 pixels

f.linha.visivel = verdade;    // configura as linhas que ligam os dados para visíveis
f.linha.cor.r = 0;            // altera a porção de cor vermelha das linhas para 0
f.linha.cor.g = 255;          // altera a porção de cor verde das linhas para 255
f.linha.cor.b = 0;            // altera a porção de cor azul das linhas para 0
f.linha.gradiente = falso;    // desativa o efeito gradiente nas cores das linhas
f.linha.iluminar = falso;     // desativa o efeito iluminação nas cores das linhas

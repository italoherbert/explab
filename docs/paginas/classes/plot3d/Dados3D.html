Classe Dados3D

Esta classe pode ser utilizada em conjunto com a classe PC3D e o comando plot3d para plotagem de um gráfico com base em dados gerados que possam formar uma curva, gráfico de função matemática ou sequência de linhas em 3D.

Atributos

Nome Tipo Valor padrão Descrição
xvetor matriz vetor Vetor de valores de x para os dados a serem plotados
yvetor matriz vetor Vetor de valores de y para os dados a serem plotados
zvetor matriz vetor Vetor de valores de z para os dados a serem plotados
legenda string null Texto com a legenda associada aos dados a serem plotados. Se valer null, a legenda é omitida no gráfico
ponto PontoConfig um objeto Permite a configuração dos pontos de cada dado. É possível configurar a visibilidade, cor e ráio da forma que é pintada se a visibilidade estiver ativada
linha LinhaConfig um objeto Permite a configuração das linhas que ligam cada dado. É possível configurar a visibilidade, a cor e os efeitos: gradiente e iluminação para cores.

Construtores

Esta classe não tem construtores definidos

Funções

setDados( matriz, matriz, matriz )

Esta função não tem valor de retorno e recebe como parâmetro os vetores com os valores de x, de y e de z. Ela seta os valores desses parâmetros aos devidos atributos. Inclusive, os vetores "xvetor" e "yvetor" e "zvetor" podem ser setados diretamente sem o uso desta função

Exemplo de uso

Este exemplo desenha um conjunto de segmentos de linhas interligados por vértices formando curvas circulares em 3D. Para tanto, são calculados os valores (x, y e z) dos vértices dos segmentos de linhas interligados.


z = [-5:0.05:5];                  // gera dados entre -5 e 5 com incremento de 0,05
                                  // e armazena em z
                                  // Isso funciona de modo parecido (mas não igual) a 
                                  // função vetesp (com exceção de a distância entre as
                                  // extremidades do intervalo serem perfeitamente 
                                  // divisiveis pelo incremento) - é o caso!

x = cos( 2*pi*z );                // gera um vetor x em função de z
y = sen( 2*pi*z );                // gera um vetor y em função de z

dados = novo Dados3D();             // instancia a classe Dados3D
dados.setDados( x, y, z );        // seta os vetores x, y e z (que têm mesma dimensão)
dados.legenda = "segmentos";      // seta a legenda do gráfico de dados

pc = novo PC3D();                   // instancia a classe PC3D
pc.graficos[] = dados;            // adiciona o objeto dados ao vetor de graficos

plot3d( pc );

Outras configurações


d = novo Dados3D();           // instancia a classe Dados2D
d.ponto.visivel = verdade;    // configura os pontos de dados para visíveis
d.ponto.cor.r = 255;          // altera a porção de cor vermelha do ponto para 255
d.ponto.cor.g = 0;            // altera a porção de cor verde do ponto para 0
d.ponto.cor.b = 0;            // altera a porção de cor azul do ponto para 0 
d.ponto.raio = 3;

d.linha.visivel = verdade;    // configura as linhas que ligam os dados para visíveis
d.linha.cor.r = 0;            // altera a porção de cor vermelha das linhas para 0
d.linha.cor.g = 255;          // altera a porção de cor verde das linhas para 255
d.linha.cor.b = 0;            // altera a porção de cor azul das linhas para 0
d.linha.gradiente = verdade;  // ativa a aplicação do efeito gradiente as cores das linhas
d.linha.iluminar = verdade;   // ativa o efeito iluminação nas cores das linhas

Expressões booleanas

Uma variável do tipo boolean pode assumir um entre dois valores possíveis: verdade ou falso. É possível definir variáveis e atribuir a elas expressões booleanas cujo resultado é sempre verdade ou falso. Veja a tabela abaixo com os operadores booleanos:

Operadores booleanos

Símbolo Descrição Exemplo Comentário
! Operador de negação - (not) !(2 > 4) Tem valor verdade, pois a afirmação: 2 > 4 é falsa (2 não é maior que quatro) e o operador ! inverte o valor da expressão que passa a valer verdade; Obs:

( !verdade ) é igual a falso
( !falso ) é igual a verdade
& Operador E - (and) !(2 > 4) & (3 > 3) Tem valor falso e envolve duas afirmações, uma do lado direito e outra do lado esquerdo. Para ser verdade a expressão booleana com duas afirmações e o operador &, ela necessita que as duas afirmações sejam verdadeiras, caso contrário, assume o valor falso. Nesse exemplo: !(2 > 4) é verdade, mas, (3 > 3) é falso. Pois, 3 não é maior que 3, e sim, igual. Obs:

( verdade & verdade ) é igual a verdade
( verdade & falso ) é igual a falso
( falso & verdade ) é igual a falso
( falso & falso ) é igual a falso
| Operador OU - (or) !(2 > 4) | (3 > 3) Tem valor verdade e envolve duas afirmações, uma do lado direito e outra do lado esquerdo. Para ser verdade a expressão booleana com duas afirmações e o operador |, ela necessita que, pelo menos, uma das duas afirmações seja verdadeira, caso contrário, assume o valor falso. Nesse exemplo: !(2 > 4) é verdade, mas, (3 > 3) é falso. Pois, 3 não é maior que 3, e sim, igual. Obs:

( verdade | verdade ) é igual a verdade
( verdade | falso ) é igual a verdade
( falso | verdade ) é igual a verdade
( falso | falso ) é igual a falso
# Operador OU Exclusivo - (xor) !(2 > 4) # (3 == 3) Tem valor falso e envolve duas afirmações, uma do lado direito e outra do lado esquerdo. Para ser verdade a expressão booleana com duas afirmações e o operador #, ela necessita necessariamente que, uma (e só uma) das duas afirmações seja verdadeira, caso contrário, assume o valor falso. Nesse exemplo: !(2 > 4) é verdade, e, (3 == 3) também é verdade. Logo o valor da expressão é falso. Obs:

( verdade # verdade ) é igual a falso
( verdade # falso ) é igual a verdade
( falso # verdade ) é igual a verdade
( falso # falso ) é igual a falso

Exemplo 1:


x = verdade;
y = falso;
z = verdade;

w = x & y | x & z;         // exibe o valor verdade, porque x & y é falso, mas x & z é verdade, e.
                               // falso | verdade é igual a verdade. Inclusive, em expressões booleanas, 
                               // as operações & são executadas primeiro 
                               // que as operações |
							   
ww = y & ( x | z );        // realiza a operação | primeiro por causa dos parenteses e o resultado é falso 
                               // porque y tem valor falso e falso & verdade é igual a falso

a = 2 > 4 & 3 != NaN;    // 2 é menor que quatro, logo: 2 > 4 é falso e 3 é diferente de NaN e, por isso, 
                               // 3 != NaN é verdade. Mas, falso & verdade é igual a falso

exibaln x;                     // exibe verdade
exibaln y;                     // exibe falso
exibaln z;                     // exibe verdade
exibaln w;                     // exibe verdade                     
exibaln ww;                    // exibe falso
exibaln a;                     // exibe: verdade
exibaln 2 < 4 & 3 != NaN;  // exibe: verdade

Exemplo 2:


a = [ 1 -2 3; 4 5 -6 ];
b = [ 1, -2, 3; 4, 5, -6 ];
c = [ -1, 3; 4, -1 ];

j = a == b | a == c;       // j recebe verdade 
k = a == b & a == c;       // k recebe falso
l = a == c;                // l recebe verdade
m = a != b;                // m recebe verdade
n = a != c;                // m recebe falso

o = a != b # a != c;       // o recebe verdade
p = a == b # a == c;       // p recebe verdade
q = a != b # a == c;       // q recebe falso                           

exibaln j;                 // exibe: verdade
exibaln k;                 // exibe: falso
exibaln l;                 // exibe: verdade
exibaln m;                 // exibe: verdade
exibaln n;                 // exibe: falso

exibaln o;                 // exibe: verdade
exibaln p;                 // exibe: verdade
exibaln q;                 // exibe: falso
